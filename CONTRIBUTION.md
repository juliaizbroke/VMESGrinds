## Contributing Guidelines

### Repository Structure
```
weekX/
├── problem-name/
│   ├── solution.py
│   ├── optimized_solution.py
│   └── problem.pdf
```

### Commit Message Format
```
week X: [problem-name] 

Example: week 1: merge strings alternately 
```

### File Naming
- Folder names: lowercase with hyphens
- Python files: `solution.py`, `optimized_solution.py`
- Problem file: `problem.pdf`
